Table Structure for CloudTik Project

Camps table columns
    Id
    Name
    Location
    Contact person
    Contact phone
    Contact email
    Mikrotik ip
    Mikrotik port
    Mikrotik username
    Mikrotik password
    Radius secret
    Radius ip
    Status
    Created at
    Updated at

Customer Type table columns
    id
    customerType (labor/cleaner/Security/Staff/Salesman/Technician/Manager)

Customers table columns
    Id
    Camp id
    Full Name
    Phone
    Email (optional)
    Username
    Password
    Status
    Created at
    Updated at

Packages
    id
    camp_id
    customerType_id
    Name
    Duration (days)
    Price (AED)
    Bandwidth
    Speed limit
    Status
    Created at
    Updated at

Counter
    id
    start amount
    end amount
    start time
    end time
    status
    user_id
    camp_id

Pagination table (optional)

Subscriptions table columns
    Id
    camp_id
    user_id
    Counter_id
    Customer_id
    Package_id
    Purchase datetime
    Subscription start
    Subscription end
    Price (AED)
    Mac address
    Status (active, pending, expired)
    Created at
    Updated at

Camp Users table
    Id
    Camp id
    User id
    Created at
    Updated at

Roles table
    Id
    Name
    User id
    Created at
    Updated at

Pages table
    Id
    Page name (camps, customers, packages, invoice)

Page access table
    Id
    Page id
    Role id
    Permission (text with json format)
    Status
